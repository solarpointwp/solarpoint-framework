# SolarPoint

SolarPoint is the modern framework to accelerate WordPress plugin development.


## Requirements

- PHP 8.1 or higher
- WordPress 6.8 or higher


## Getting Help

Get support from the community by browsing existing discussions or asking a question on [GitHub Discussions](https://github.com/solarpointwp/solarpoint-framework/discussions/categories/q-a).

Detailed instructions for reporting bugs, requesting features, and more can be found in our [support documentation](SUPPORT.md).


## Code of Conduct

In order to keep SolarPoint welcoming to everyone, please read and follow our [Code of Conduct](https://github.com/solarpointwp/solarpoint-framework?tab=coc-ov-file).


## Security Vulnerabilities

Please review [our security policy](https://github.com/solarpointwp/solarpoint-framework/security/policy) for information on how to report security vulnerabilities.
